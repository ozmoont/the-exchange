# Specs

This folder holds one markdown file per feature spec, written by **Franko** before any implementation starts.

Naming: `<short-slug>.md` (e.g. `real-icabbi-adapter.md`, `fee-override-editor.md`).

Each spec follows the shape Franko enforces:
- **Problem** — 1–2 sentences, who hurts, why now
- **Out of scope** — the things explicitly NOT being solved
- **Acceptance criteria** — 3–7 numbered, behaviour-described, independently verifiable
- **Files likely touched** — Mykola refines during implementation
- **Risks / open questions**

See `.claude/agents/franko.md` for the full process.
